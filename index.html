Tentang saya

Permisi, perkenalkan saya adalah seorang mahasiswa semester 6 prodi Sistem Informasi di Universitas Alma Ata Yogyakarta.

Sebagai mahasiswa yang sedang menekuni bidang ilmu tegnologi infomasi terutama dalam management organization dan pengembangan web development, saya sangat berminat untuk meningkatkan soft skill dan juga hardskill dalam bidang tersebut dengan mengikuti program studi kampus, course online, serta IT project.

Saya memiliki kemampuan dasar mengenai perancangan web seperti HTML, CSS, JAVASCRIPT. Saya yakin dengan ketekunan dan konsisten yang rutin serta minat dan tekad untuk mendalami bidang tegnology information, mampu menjadi bekal langkah awal saya untuk bisa tumbuh dan berkembang menjadi bagian web development.
